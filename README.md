## Deskripsi Aplikasi
Aplikasi kuis interaktif berbasis Flutter yang dirancang untuk menguji pengetahuan pengguna melalui pertanyaan pilihan ganda. 

User dapat:
- Menjawab serangkaian soal
- Melihat progres secara real-time
- Mendapatkan hasil akhir setelah semua pertanyaan dijawab
